# oop-hw1

## Kратко описание на класовете
Author: Представя автор с име и година на раждане.
Book: Моделира книга с автор, година, цена и ISBN и следи общия брой книги.
Member: Представя читател с име, идентификатор и година на присъединяване.
Loan: Управлява заем на книга с дати и статус върната/не върната.
Library: Координира книги, членове и заеми и предоставя основна библиотечна логика.

## как се компилира
g++ -std=gnu++17 -Wall -Wextra -O2 -Iinclude src/main.cpp -o library
library.exe - за да се run-не
